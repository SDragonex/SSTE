# 💻 Školní projekty – SŠTE Brno

Tento repozitář obsahuje moje školní projekty vytvořené během studia na **Střední škola technická a ekonomická Brno (SŠTE Brno)**.

Projekty slouží k procvičení webových technologií a programování.

---

## 🛠 Použité technologie

* 🌐 HTML5
* 🎨 CSS3
* ⚙️ JavaScript
* 🐘 PHP
* 🗄️ (případně MySQL – pokud používáš databázi)

---

## 📂 Obsah repozitáře

Repozitář obsahuje menší projekty, například:

* Statické webové stránky
* Responzivní layouty
* Formuláře s validací
* JavaScript interaktivní prvky
* Jednoduché PHP aplikace
* Práce s databází (pokud máš)

---

## 🎯 Cíl projektů

* Procvičit tvorbu webových stránek
* Naučit se základy frontend a backend vývoje
* Pochopit práci se serverem a formuláři
* Zlepšit strukturu a přehlednost kódu

---

## 🚀 Spuštění projektů

### HTML/CSS/JS projekty

Stačí otevřít `index.html` v prohlížeči.

### PHP projekty

Je potřeba lokální server, například:

* XAMPP
* WAMP
* nebo vestavěný PHP server
